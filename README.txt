Detection_argumentations_in_russian_texts_by_transformers

1. Файл dataset1.xlsx содержит таблицу из двух столбцов: 1)тексты содержащие и не содержащие аргументацию, 2) (0 или 1) 1 указывает на наличие аргументации, 0 - на её отустствие
Корпус состоит из 1400 текстов, а именно 700 аргументационных и 700 неаргументационных. Соответственно корпус сбалансирован. 80% корпуса было использовано для обучения моделей,
20% для тестирования. 
2. learn.py - скрипт обучения модели классифицировать тексты по наличую аргументации, используя для этого dataset1.xlsx
3. test_model.py - скрипт тестирования модели, нахождения метрик (accuracy, error rate, precision, recall, F-measure). Необходимо для сравнения дообученных моделей на основе RuBERT и RuRoBERTa классифицировать тексты согласно нашей задаче.
